# Executive Sales Dashboard

Dashboard desenvolvido em Power BI para análise de vendas e lucro.

## Objetivo
Auxiliar gestores na tomada de decisão utilizando visualização de dados.

## Tecnologias
- Power BI
- CSV
- Power Query

## Funcionalidades
- KPI de faturamento
- KPI de lucro
- Produtos mais vendidos
- Lucro por região
- Dashboard interativo

## Insights
- Região Sudeste apresentou maior lucro.
- Notebook foi o produto com maior faturamento.
- Categoria Informática lidera vendas.
